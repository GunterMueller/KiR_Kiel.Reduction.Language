Download Example Programs

This programs are KiR programs and therefore coded. To see their readable notation you must load them into the KiR editor.
INTER.ed
Library of advanced I/O functions.
STRING.ed
Library of string manipulation functions.
io_dir.ed
Asks the user for a directory name and returns the list of directory entrys.
io_firstline.ed
Returns the first line of a file.
io_filio.ed
Header for file manipulation. Uses INTER.ed.
io_ggt.ed
Asks the user for a list of numbers and prints the greatest common divisor. Uses INTER.ed.
io_ll.ed
Asks the user for a directory name and prints the directory entrys to screen like 'ls -l'. Uses INTER.ed and STRING.ed.
